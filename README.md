# 📊 Customer Segmentation & Retention Dashboard

## 🚀 Overview

A Power BI dashboard to analyze customer behavior, segmentation, and retention. Helps identify high-value customers, churn risk, and growth opportunities.

---

## 🎯 Objective

* Analyze customer segments
* Track retention & churn
* Improve decision-making

---

## 🛠️ Tools Used

* Power BI
* Excel
* DAX



## 📊 Features

* KPI Tracking (Customers, Retention %, Churn %)
* Segment-wise analysis
* Retention trends
* Interactive filters
